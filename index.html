Applications I've Created

Letter Art >>

Letter Art

The Letter Art application is one that I created using PHP, jQuery and MySQL. It allows the user to type in a word and pops up pictures to spell out the word. The user can then click on the letter images to change them to a different image for the same letter.

Tic Tac Toe >>

Tic Tac Toe

A simple Tic Tac Toe game where people can play back and forth on the same computer, clicking on squares in the grid to make Xs and Os until someone wins or there is a tie and a new grid will come up for them to play on. This was created using CSS and jQuery.
